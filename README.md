# Awesome Angular 2+ Libraries and Tools üöÄ

This repository aims to provide a comprehensive list of libraries compatible with Angular 2 and higher versions. Finding Angular libraries can be challenging due to older Angular 1 results. Here, you'll find a curated, categorized, and well-organized list of Angular libraries, tools, modules, and starter projects to supercharge your development! Whether you're building a design-heavy application, optimizing performance, or starting a new project, this collection has you covered. Contributions are always welcome! **Have a library or tool to add? Open a Pull Request or create an Issue! üéâ**
